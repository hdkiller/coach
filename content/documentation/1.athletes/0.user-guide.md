## Daily home base

| Area              | Open              | Use it to                                                             |
| ----------------- | ----------------- | --------------------------------------------------------------------- |
| **Dashboard**     | **Dashboard**     | Morning check-in, wellness snapshot, training load, sync, and upload  |
| **Chat**          | **AI Chat**       | Ask questions, adjust plans, analyze sessions, and run approved tools |
| **Activities**    | **Activities**    | Calendar and list of planned and completed workouts                   |
| **Training Plan** | **Training Plan** | Active periodized plan, blocks, and adaptations                       |

Start most days on the **Dashboard**, complete check-in when prompted, then open **Chat** or **Activities** for the session ahead. See [Dashboard & Daily Flow](/documentation/athletes/dashboard-daily-flow) and [AI Chat & Supported Tools](/documentation/athletes/chat-interaction).

## Plan and complete training

1. **Build or apply a plan** — **Training Plan** or **Library → Plans**. See [Training Plans](/documentation/athletes/training-plans).
2. **Schedule sessions** — from **Activities** calendar (workout drawer) or by applying library templates. See [Activity Calendar & Workout Management](/documentation/athletes/activity-management).
3. **Train** — complete outdoors, indoors, or in a connected strength app.
4. **Sync or upload** — Dashboard **Sync**, connected apps, or `.fit` upload.
5. **Match and review** — link completed work to planned sessions; open the workout detail for scores, intervals, strength sets, and sharing. See [Activities & Workout Analysis](/documentation/athletes/activities-workouts) and [Workout Detail Guide](/documentation/athletes/workout-details).

![Activities calendar with completed, planned, and missed sessions plus weekly plan versus actual totals](/media/docs/activities-calendar.webp)

For gym work, see [Strength Training](/documentation/athletes/strength-training). For duplicates across Garmin and Strava, see [Duplicate Workouts & Versions](/documentation/athletes/duplicate-workouts).

## Libraries

**Library** holds reusable content separate from dated calendar items:

- **Workouts** — endurance and gym templates
- **Exercises** — strength building blocks
- **Plans** — multi-week blueprints
- **Charts** — saved analytics views

See [Workout, Exercise & Plan Libraries](/documentation/athletes/library).

## Working with a human coach

If you have a coach:

1. Generate or accept a connection from **Coaching → My Coaches** (athlete side) using the flow your coach prefers.
2. Your coach can view permitted training data and schedule work on your calendar.
3. Revoke access anytime from your coaching connections.

Coaches do not need your password. See the coach docs on [Connecting with Athletes](/documentation/coaches/connecting-athletes) if you are helping someone set up.

## Settings checklist

Open **Settings** (and **Profile → Settings**) for:

| Section        | Purpose                                   |
| -------------- | ----------------------------------------- |
| Basic profile  | Name, timezone, units, language           |
| Sport profiles | FTP, LTHR, pace/power/HR zones            |
| Availability   | Days and windows for plan generation      |
| Nutrition      | Fueling sensitivity and targets           |
| AI Coach       | Personality, tools, quotas                |
| Apps           | Connections, ingestion, source preference |
| Billing        | Plan, portal, entitlements                |
| Data / Danger  | Export, category wipe, account deletion   |
| Developer      | API keys and OAuth apps                   |

Full index: [Settings](/documentation/athletes/settings). Account help: [Account & Billing](/documentation/help/account-billing).

## Metrics in one sentence

Coach Watts combines classic load metrics (TSS, CTL, ATL, Form) with AI **1–10 scores** on workouts, nutrition, and reports. Definitions live in the [Glossary](/documentation/help/glossary) and [Metrics & Scoring](/documentation/athletes/metrics-scoring).
