Coach Watts has several analytics surfaces. Use **Performance** for guided trends and use the tools below for custom investigation.

## Chart library and Analytics Explorer

Open **Library → Charts** to browse saved visualizations. Select a chart to open it in Analytics Explorer, where you can change the visible date range and inspect its values.

Available actions include:

- Open the underlying explorer
- Edit the visualization source
- Pin a visualization to a dashboard
- Reuse a saved chart instead of rebuilding it

## Build a visualization

Open the chart builder from the analytics tools.

1. Enter a visualization name.
2. Choose a data source and chart type.
3. Set the time range and grouping.
4. Add one or more metrics.
5. Choose an aggregation for each metric.
6. Save the visualization to the library.

Choose grouping and aggregation together. For example, summing weekly duration answers a different question than averaging individual workout duration.

## Workout Explorer

Use Workout Explorer for a deep look at one selected workout. Depending on the source data, it can show summary information, intervals, streams, and density or distribution views.

Workout Explorer is best for investigating a single session without changing the layout of the main workout page.

## Add workouts to comparison

From a workout page, select **Add to Comparison**. Add the other sessions the same way, then open **Workout Comparison**.

The comparison basket persists while you navigate. Remove a workout from the basket when it no longer belongs in the analysis.

## Configure a comparison

Workout Comparison supports several views and controls, including:

- Chart type and sort order
- Primary and secondary summary metrics
- Stream field
- Stream alignment
- Interval metric

Alignment matters when workouts have different durations. Match by elapsed position or the available normalized alignment when you want to compare the shape of efforts rather than identical timestamps.

## Interpret results carefully

- Compare like sports and similar workout purposes where possible.
- Check units and sport settings before comparing pace or zones.
- Missing source streams can make two summary-identical workouts look very different.
- A higher value is not always better; training purpose determines the interpretation.
