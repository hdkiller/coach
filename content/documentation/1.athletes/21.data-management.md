title: Data Management

Open **Data Management** from the application search or developer settings when you need to inspect the records behind dashboards and analyses.

## What the page contains

Use the section buttons to move between:

- Summary
- Fitness and recovery records
- Planned workouts
- Racing events
- Completed workouts
- Nutrition data

This view is useful when a chart appears wrong, a sync created an unexpected record, or you need to remove a specific item without clearing an entire integration.

## Delete an individual record

1. Find the record in its section.
2. Check the date, source, and identifying values.
3. Select the delete action.
4. Confirm the warning.

Deletion is permanent for the Coach Watts record. If the source integration remains connected, a later sync may import the source record again.

## Removing duplicates

Do not delete the lower-quality version merely to clean up calendar totals. Use [Duplicate Workouts & Versions](/documentation/athletes/duplicate-workouts) so Coach Watts retains source attribution and can preserve useful data.

Delete only when the record itself should no longer exist in Coach Watts.

## Disconnecting an app versus deleting data

- **Disconnect or revoke** stops future access or synchronization.
- **Delete a record** removes one Coach Watts record.
- **Wipe imported data** removes an entire category of imported records.
- **Delete account** removes the account and is irreversible.

Disconnecting an app does not automatically remove its existing data.

## Bulk and destructive cleanup

Open **Settings → Danger Zone** for category-wide actions such as clearing planned schedules, synced activities, wellness data, nutrition logs, AI analysis, or generated athlete profiles.

Use those controls only after reading their confirmation carefully. Some schedule actions can also affect Intervals.icu.

## Export your data

Open **Settings → Danger Zone → Data Portability** and select **Export My Data**. The export is a JSON archive containing data such as workouts, health metrics, and chat history.
