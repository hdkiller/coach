title: Strength Training
description: Build gym exercises and strength workouts, schedule them on your calendar, and bring completed sessions into Coach Watts.
---

Coach Watts treats strength work as first-class training load: reusable **exercises**, **Gym** workout templates with blocks and sets, scheduled planned sessions, and completed activities that can match those plans.

## What you can do

| Step                   | Where                                                                                 |
| ---------------------- | ------------------------------------------------------------------------------------- |
| Define moves           | **Library → Exercises**                                                               |
| Build a gym session    | **Library → Workouts** → type **Gym**                                                 |
| Put it on the calendar | **Activities** → workout drawer → schedule template                                   |
| Log the session        | Prefer [Hevy](/documentation/athletes/integrations/hevy), or upload / manual activity |
| Review                 | Workout detail (strength exercises tab) + match to planned                            |

## Create exercises

1. Open **Library → Exercises**.
2. Select **New Exercise**.
3. Set title, movement pattern (for example squat, hinge, push, pull, lunge, core, carry, mobility), intent, muscle groups, and prescription defaults (reps, duration, load mode).
4. Add optional instructions or video information.
5. Save.

Use filters (pattern, intent, muscle groups) and search to find exercises when building sessions. Deleting an exercise does not rewrite historical completed workouts that already recorded it.

![Exercise library with filters for movement pattern, intent, and muscle groups](/media/docs/library-exercises.webp)

## Build a strength workout template

1. Open **Library → Workouts**.
2. Create a **New Workout** with type **Gym** (weight training).
3. Open the strength editor and add **blocks** such as warmup, single exercise, superset, circuit, or cooldown.
4. **Add Exercise** from your library (or create inline), then configure set rows, rest, and load fields the editor supports.
5. Optionally save an inline exercise back to the library.
6. Set duration or TSS override when you want planned load to reflect the session.

Endurance structured workouts (power/HR/pace intervals) use a different builder — see your workout library and planned workout detail for those sports.

![Gym workout template detail with strength execution plan, blocks, exercises, and set rows](/media/docs/library-gym-workout.webp)

## Schedule a session

1. Open **Activities** in calendar view.
2. Open the workout drawer and choose your Gym template.
3. Place it on the target date and confirm **Schedule Workout**.
4. Open the planned workout to verify targets before training day.

Coaches can also place templates on your calendar from **Coaching → Calendar**.

::alert{type="info"}
Scheduling is calendar-driven. Prefer the Activities (or coach calendar) drawers rather than expecting every library list button to place a session.
::

## Log and complete strength work

Coach Watts does not currently provide a live “set logger” while you lift. Practical paths:

1. **Hevy (recommended)** — Connect **Settings → Apps → Hevy**, complete the session in Hevy, then sync. Strength volume feeds fatigue and recovery context. See [Hevy Integration](/documentation/athletes/integrations/hevy).
2. **`.fit` upload** — Dashboard **Upload** or **Workouts → Upload** when you have a file.
3. **Manual activity** — Create or edit an activity with a strength/gym type when you only need a calendar placeholder.

After a completed session appears:

1. Open **Activities**.
2. Drag or use match tools to link it to the planned Gym workout when they belong together.
3. Open the completed workout detail to review strength exercises, scores, and notes.

See [Activity Calendar & Workout Management](/documentation/athletes/activity-management) and [Duplicate Workouts & Versions](/documentation/athletes/duplicate-workouts) if the same session synced from multiple sources.

## Strength and training load

Strength sessions contribute to overall fatigue and recovery recommendations. Keep sport profiles and availability accurate so plan generation leaves room for gym days — [Measurements & Availability](/documentation/athletes/settings/measurements-availability).
