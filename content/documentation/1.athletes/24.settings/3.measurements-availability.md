Both areas are under **Settings → Profile**.

## Add a measurement

Open the **Measurements** tab and select a metric. Enter the value, unit, recorded time, and optional note. Custom measurements can be created when the predefined list does not cover your use case.

Use the recorded date rather than today's date when entering historical data. Trends and downstream calculations depend on measurement timing.

## Edit and filter history

Measurement history can be filtered by metric and source. Edit a record to correct its value or note. Do not edit a provider record merely to change the current source preference; configure preferred sources instead.

## Preferred measurement source

When multiple services provide the same metric, select the preferred source. This helps Coach Watts choose a consistent value instead of alternating between devices with different measurement methods.

Changing preference does not delete the other source records.

## Configure weekly availability

Open the **Availability** tab. For each available day:

1. Add one or more sessions.
2. Set the start time and maximum duration.
3. Choose the permitted activity types.
4. Mark **Indoor Only** when outdoor sessions are not possible.
5. Save the weekly schedule.

Multiple sessions allow split days, such as a morning ride and evening strength session. Leave a day empty when it should normally remain unavailable.

## How plans use availability

Availability is a planning constraint, not a guaranteed appointment. The AI considers goal priority, recovery, session sequence, and required volume before filling every available slot.

If plans repeatedly schedule impossible sessions, check:

- Timezone and day boundaries
- Session duration
- Allowed activity types
- Indoor-only restrictions
- Whether multiple sport profiles map to the same activity
