---
title: Basic Profile & Localization
description: Configure identity, body metrics, languages, timezone, and display units.
---

Open **Settings → Profile → Basic Settings**.

## Personal information

Set your name, sex, and date of birth. These values can affect profile presentation and physiological or metabolic calculations. Your sign-in email is displayed but managed through the account identity rather than this form.

## Body metrics

Set current weight and height with their units. Weight affects calculations such as carbohydrate targets per kilogram. For historical measurements and synced-source priority, use the **Measurements** tab instead of repeatedly overwriting the current value.

## Language settings

- **UI language** changes application labels where translations are available.
- **AI language** tells the coach which language to prefer in generated responses and analysis.

Changing the UI language does not translate historical free-text notes or previously generated analyses.

## Timezone

Timezone controls how UTC timestamps are grouped into local calendar days. Set it to the location where you normally train and review data.

Changing timezone can move an overnight or late-evening record to a different displayed day. It does not change the source recording's actual timestamp.

## Units

Configure weight, height, and distance units. Unit settings change display and data entry; normalized stored data and source files remain intact.

## Auto-detect profile information

Use **Auto-detect** to propose supported values from recent data. Review every suggestion before applying it. Auto-detection is an aid, not a replacement for a recent lab or field test.
