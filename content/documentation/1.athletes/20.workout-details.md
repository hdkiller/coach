title: Workout Detail Guide
description: Analyze, organize, share, compare, export, and correct a completed workout.
---

The workout detail page combines the source recording, derived metrics, plan adherence, AI analysis, and tools for organizing the activity.

## Navigate workout sections

Available sections depend on the sport and source data. They can include:

- Overview, strength exercises, and workout nutrition
- AI analysis and plan adherence
- Power curve, intervals, advanced metrics, map, and pacing
- Timeline, zones, efficiency, notes, metrics, and streams
- Versions and raw source data

Use the section toolbar to jump directly to a section. Open the section settings to hide sections you do not use or change their order. Unavailable sections remain hidden even when enabled.

## Edit workout information

Use the workout actions to edit supported summary information. Local tags can be added or removed on the workout. Tags imported from Intervals.icu are displayed separately and remain read-only in Coach Watts.

## Re-run analysis

Use **Analyze** when analysis is missing or should be regenerated after correcting important workout or profile information. Analysis consumes the current sport settings, thresholds, and available source data.

If the workout is linked to a planned session, regenerate **Plan Adherence** separately when its match or plan structure changed.

## Manage the planned-workout link

The plan-adherence section shows the planned workout used for comparison.

- **Regenerate** recalculates adherence against the linked plan.
- **Unlink** keeps both records but removes the completed-to-planned relationship.

To choose a different planned workout, return to **Activities → Link Workouts** after unlinking.

## Manage duplicate versions

The **Versions** section appears when the workout is a duplicate, has linked duplicates, or has a planned-workout relationship.

- Open the primary or secondary version.
- Promote a secondary source to primary.
- Unlink a version that represents a different session.

See [Duplicate Workouts & Versions](/documentation/athletes/duplicate-workouts) before changing the primary.

## Save to the workout library

Select **Save to Library** to create a reusable workout template from the session. Review the saved template before scheduling it because recorded execution data and a reusable prescription serve different purposes.

## Compare workouts

Select **Add to Comparison**. Repeat for the other workouts, then open **Workout Comparison** to compare summary metrics, intervals, or aligned streams. The comparison basket persists while you browse.

See [Analytics Explorer & Workout Comparison](/documentation/athletes/analytics-tools).

## Share a workout

Open **Share**, choose the available content and expiry option, then create the link. Anyone with an active public link can view the shared page, so inspect it before sending.

Create a new link when you need different sharing options. Do not treat an unexpired public link as private authentication.

## Export and publish

Depending on the source and available data, workout tools can include:

- Downloading a FIT representation
- Viewing route and stream data
- Publishing the AI summary to Intervals.icu when that integration supports it

Publishing a summary sends the current analysis outward; it is different from syncing the original workout into Coach Watts.

## Raw data

Raw data is intended for troubleshooting source fields and integration behavior. Values can use provider-specific names or units and may differ from normalized metrics shown elsewhere on the page.
