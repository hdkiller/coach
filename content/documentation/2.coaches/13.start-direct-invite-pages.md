title: Start & Direct Invite Pages
description: Build a public intake funnel and customize the branded experience around coach-generated join links.

Coach Watts has two public onboarding experiences. Configure both under **Profile → Settings → Public Presence**.

## Know the difference

| Page                   | Route pattern         | Outcome                                                         |
| ---------------------- | --------------------- | --------------------------------------------------------------- |
| **Coach Start Page**   | `/coach/{slug}/start` | Athlete submits an intake request for you to approve or decline |
| **Direct Invite Page** | `/join/{code}`        | Athlete accepts an invitation you already generated             |

The public coach profile is separate again: it is your marketing/profile minisite and can link visitors toward either next step.

## Configure the Coach Start Page

1. Set and save your coach slug in **Coach Profile**.
2. Turn on **Enable coach start page**.
3. Save, preview, then select **Edit Start Page**.

The editor supports ordered, hideable sections for:

- hero and introductory copy;
- expectations and what happens next;
- trust/proof;
- pricing and offers;
- no-commitment reassurance;
- FAQ;
- intake form;
- final CTA.

Pricing offers can include a name, price label, cadence, description, features, and an optional external CTA. The intake form can customize its title, intro, submit and login labels, success message, questions, help text, and select options.

## Handle submitted requests

Submitted forms appear in **Coaching → Athletes → Pending Coaching Requests**. Review the answers and any notice about the athlete's existing coaches, then select **Approve** or **Decline**.

Submitting the form does not automatically grant access to athlete data. Approval creates the coaching relationship.

## Configure the Direct Invite Page

Turn on **Enable branded direct invite page**, then customize:

- join headline, intro, and CTA label;
- welcome title and rich-text copy;
- trust title and note;
- join steps;
- FAQ;
- the fallback note shown when no active invitation is available.

This content wraps coach-generated share and email invitation links. The invitation's code, email restriction, status, and expiry still determine whether the athlete can join.

## Test before sharing

1. Preview each page.
2. Check mobile and desktop layout.
3. Submit a test request and verify it appears in the roster review queue.
4. Generate a fresh direct invitation and test its join page.
5. Revoke test invitations when finished.
