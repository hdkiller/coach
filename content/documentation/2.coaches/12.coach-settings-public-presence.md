Open **Profile → Settings**. Coach accounts use the same core profile tabs as athletes, plus coach-specific publishing tools under **Public Presence**.

## Settings tabs

| Tab                 | What it controls                                                                     |
| ------------------- | ------------------------------------------------------------------------------------ |
| **Basic Settings**  | Name, nickname, language, units, location, timezone, visibility, and profile basics  |
| **Sport Settings**  | Sport profiles, activity-type matching, thresholds, and zones used by training tools |
| **Measurements**    | Body measurements and measurement history                                            |
| **Availability**    | Weekly availability and time windows used by planning tools                          |
| **Nutrition**       | Nutrition preferences and calculation inputs                                         |
| **Public Presence** | Coach profile, coach start page, direct-invite page, and optional athlete profile    |
| **Communication**   | Communication preferences                                                            |

For the detailed shared settings reference, see [Athlete Settings](/documentation/athletes/settings). The data and units you configure affect your own account; reviewing an athlete does not overwrite their settings.

## Enable a coach profile

In **Public Presence → Coach Profile**:

1. Turn on **Enable coach page**.
2. Choose a unique coach slug.
3. Set display name, headline, and coaching brand.
4. Add an optional primary CTA URL.
5. Set the SEO title and description.
6. Save, preview, then select **Edit Page** for full page content.

The public coach route uses your slug. Disabling the page takes the public profile out of service without deleting your saved content.

## Edit the public coach page

Edit mode lets you reorder, hide, and configure sections including hero, credibility, about, specialties, credentials, FAQ, testimonials, gallery, video introduction, featured plans, all plans, social/contact blocks, and a final CTA.

Only enabled sections render publicly. Preview the page while signed out or in a private window before sharing it.

## Start and direct-invite settings

The same tab contains:

- **Coach Start Page** — a public coaching-request funnel with an editable intake form.
- **Direct Invite Page** — branded content displayed around raw `/join/:code` invitations.

These pages serve different purposes. The start page collects a request for your approval; a direct invite accepts an already-created invitation.
