The **Analytics** area is shared by athletes and coaches. Coaches with connected athletes can scope supported charts to one athlete or a multi-athlete selection.

## Analytics tools

| Tool                 | Purpose                                                                  |
| -------------------- | ------------------------------------------------------------------------ |
| **Dashboards**       | Arrange saved charts into named tabs with a shared default scope         |
| **Chart Explorer**   | Browse system charts and preview them against personal or athlete scopes |
| **Chart Builder**    | Choose a source, chart type, time range, grouping, metrics, and scope    |
| **Workout Explorer** | Inspect workout summaries, intervals, streams, density, and maps         |
| **Workout Compare**  | Compare selected completed workouts and their metrics or streams         |

## Select athlete scope

In **Chart Explorer**, open the roster selector:

1. Choose **Single** and select one athlete for a focused review.
2. Choose **Compare** and select multiple athletes for supported squad charts.
3. Preview the chart, then select **Pin to Dashboard** if you want to keep it.

Some visualizations only support personal or single-athlete data. The interface warns when a chosen chart cannot use the current scope.

## Dashboard scope and widget scope

A dashboard tab has a default scope. Widgets can either:

- **inherit** the tab's scope; or
- **override** it with their own athlete or athlete-set scope.

This lets one team dashboard follow a shared roster selection while a specific recovery or comparison chart remains fixed to selected athletes.

## Compare workouts from the coach calendar

1. Open **Coaching → Calendar**.
2. Hover a completed workout and select **Add to comparison**.
3. Add more completed workouts from either athlete lane.
4. Open the comparison dock.
5. Choose chart type, metrics, stream field, alignment, interval metric, and sort order as needed.

## Custom metrics

Use **Manage Metrics** to define numeric measurements that are not part of the standard system fields. Set their scope, label, key, data type, and unit. Only numeric custom metrics are chartable at present.

## Interpreting coach data

Use the athlete profile for a fast operational summary and Analytics for deeper investigation. Compare like-for-like periods and sports, and treat readiness, compliance, CTL, ATL, and TSB as decision inputs rather than stand-alone prescriptions.
