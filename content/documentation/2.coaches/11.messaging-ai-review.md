---
title: Messaging & AI-Assisted Review
description: Understand the current Message Athlete action and use AI chat safely during a coach review.
---

Athlete cards and athlete profiles include a message action, but its current behavior is important: it opens **Coach Watts AI Chat** with a starter prompt about the athlete. It does **not** send a direct message, email, or notification to the athlete.

For the complete conversation, media, memory, approval, and tool reference, see [AI Chat & Supported Tools](/documentation/athletes/chat-interaction).

## Use the message action

1. Select the message icon on an athlete card or athlete profile.
2. Coach Watts opens `/chat` with a prompt such as “I'd like to discuss [athlete]'s training. What should we focus on?”
3. Edit the prompt so it asks a precise coaching question.
4. Review the answer against the athlete profile, calendar, and analytics.

## Useful review questions

- “Summarize the main compliance and recovery signals I should inspect.”
- “What data would help distinguish accumulated fatigue from one poor session?”
- “Give me a checklist for reviewing the athlete's next seven days.”

The athlete's name in the starter prompt is conversational context; it is not proof that every private athlete field has been attached to that chat turn. Verify important claims in the coach-visible data screens.

## Contact the athlete

Use your established external communication channel when you need to contact the athlete directly. Do not assume the in-app message icon has delivered anything to them.

## Good practice

- Avoid pasting unnecessary sensitive information into chat.
- Treat AI output as analysis support, not an automatic training decision.
- Confirm schedule changes in the coach calendar.
- Discuss material changes with the athlete through your normal communication channel.
